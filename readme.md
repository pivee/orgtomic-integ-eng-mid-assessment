## Description

This is a simple REST API built following the instructions given in [Assessment Guidelines](./docs/INSTRUCTIONS.md) to integrate with [indeed.com](indeed.com).

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
